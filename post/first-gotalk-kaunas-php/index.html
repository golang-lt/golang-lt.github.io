First gotalk in Kaunas PHP

Sun, Nov 16, 2014

We will consider a birth of gophers.lt golang user group started in Kaunas PHP meetup, since the gotalk introducing the user group was held there.

The talk was an introduction to golang and the main features: concurrency primitives, interfaces and composition. You can find it here.

Community

So far we are only starting to organize an user group. And will post all upcomming events even if it’s only a beer meetup. Stay put and subscribe to the RSS feed, join us on facebook or subscribe to our mailing list.

Plans

We are planning to organize some events, give talks and based on the interest, maybe open a monthly conferences here in Kaunas DATA-DOG headquarters.
